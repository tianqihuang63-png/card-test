# 个人介绍卡片生成器

一个绿色清新风格的个人名片生成器，用户填写信息后可实时预览并下载竖向名片的 PNG 图片。

## 功能特性

- ✨ **绿色清新 UI 设计**：渐变背景，现代化风格
- 📝 **实时预览**：输入即所见，即时更新卡片效果
- 📷 **头像上传**：支持上传个人头像照片
- 🎨 **竖向名片**：400×600 像素，适合分享和打印
- 📱 **响应式设计**：支持桌面、平板和移动设备
- 💾 **PNG 导出**：一键下载高清图片
- 📲 **二维码生成**：自动生成包含联系信息的二维码
- 🚀 **零依赖**：纯 HTML/CSS/JavaScript，无需构建工具

## 快速开始

### 方式一：直接打开（推荐）

1. 双击 `index.html` 文件，用浏览器打开
2. 填写您的个人信息
3. 上传头像（可选）
4. 点击"下载名片"按钮
5. 获得 PNG 格式的个人介绍卡片

### 方式二：本地服务器

```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js
npx serve

# 然后访问 http://localhost:8000
```

## 技术栈

- **HTML5**：页面结构
- **CSS3**：绿色渐变主题、响应式布局
- **JavaScript (ES6+)**：交互逻辑
- **html2canvas**：将 DOM 转换为 Canvas
- **qrcode.js**：生成二维码
- **Font Awesome**：图标库

## 输入字段

| 字段 | 说明 | 必填 |
|------|------|------|
| 头像 | 上传个人头像照片 | 否 |
| 姓名 | 您的姓名 | 是 |
| 职位/头衔 | 您的职位或头衔 | 否 |
| 公司 | 所在公司名称 | 否 |
| 个人简介 | 简短的自我介绍（最多100字） | 否 |
| 邮箱 | 联系邮箱 | 否 |
| 电话 | 联系电话 | 否 |

## 卡片规格

- **尺寸**：400 × 600 像素
- **格式**：PNG
- **分辨率**：2x（导出时自动提升）
- **二维码内容**：`姓名 | 职位 | 公司 | 邮箱 | 电话`

## 颜色方案

- 主色：#4CAF50（清新绿）
- 辅助色：#81C784（浅绿）
- 背景渐变：#E8F5E1 → #A5D6A7
- 文字色：#2E7D32（深绿）

## 浏览器兼容性

- Chrome/Edge (推荐)
- Firefox
- Safari
- Opera

## 注意事项

1. 头像图片大小限制：5MB
2. 支持的图片格式：JPG、PNG、GIF、WebP
3. 个人简介最多 100 字符
4. 导出需要至少填写姓名字段
